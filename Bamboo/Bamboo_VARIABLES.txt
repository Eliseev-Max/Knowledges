• Переменные сборки (Build-specific variables) динамически оцениваются Bamboo во время сборки. 
Источником переменной сборки может быть либо свойство Bamboo, либо один из подключаемых модулей по умолчанию (при условии, что он включен).
    
• Переменные развертывания (Deployment variables) доступны при развертывании проекта.
• Системные переменные также применяются ко всему экземпляру Bamboo и наследуют свои значения от системных переменных или переменных среды с тем же именем.
• Глобальные переменные определяются для всего экземпляра Bamboo и имеют одинаковое (статическое) значение для каждого плана, созданного Bamboo.
    (См. Определение глобальных переменных)
	
• Переменные проекта определяются для конкретных проектов.
	Переменные проекта могут переопределять глобальные переменные с тем же именем.
	(См. Определение переменных проекта)
• Переменные плана аналогичны глобальным переменным, но определяются для конкретных планов.
	Переменные плана переопределяют глобальные переменные и переменные проекта с теми же именами. Вы также можете переопределить переменную плана для сборки, если запускаете сборку вручную.
	(См. Определение переменных плана)
