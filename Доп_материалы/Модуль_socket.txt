Модуль socket обеспечивает доступ к интерфейсу сокета BSD.
Он доступен во всех современных ОС: Unix, Windows, MacOS

Сокет - это программный интерфейс для обеспечения информационного обмена между процессами

Сокеты:
	• серверные ► прослушивают определённый порт
	• клиентские ► подключаются к серверу

После установления соединения начинается обмен данными
Создаётся приложение, которое прослушивает свой порт

В большинстве операционных систем прослушивание портов с номерами 0 — 1023 требует особых привилегий

Интерфейс Python представляет собой прямую трансляцию системного вызова Unix и интерфейса библиотеки для сокетов
в объектно-ориентированный стиль Python.
Функция socket.socket() возвращает объект Socket, методы которого реализуют различные системные вызовы сокетов.
(Типы параметров функций модуля несколько более высокоуровневые, чем в интерфейсе языка C)

socket.socket(family=AF_INET, type=SOCK_STREAM, proto=0, fileno=None)

Метод bind():
	► связывает сокет с хостом и портом;
	► принимает в качестве аргумента:
		кортеж вида  (IP_ADDRESS, PORT)

Метод listen():
	► запускает для сокета режим прослушивания
	► принимает аргумент: int() = максимальное количество подключений в очереди

Метод accept():
	► принимает подключение;
	► не принимает никаких аргументов;
	► возвращает кортеж (новый_сокет, адрес_клиента)

Метод recv():

Метод send():
	► Отправляет байтовую строку
	► принимает байты (b"string")

Метод conn.close()
	► метод от первого элемента кортежа, возвращаемого методом .accept()
	► закрывает соединение
