Последовательности в Python (унаследованы от ABC).
Общий набор операций имеют:
	• строки
	• списки
	• последовательности байтов
	• массивы
	• элементы XML
	• результаты выборки из БД
Набор операций включает в себя:
	♦ итерирование;
	♦ получение среза;
	♦ сортировка;
	♦ конкатенация.

Встроенные последовательности:

	► контейнерные (list, tuple, collections.deque)
		в контейнерных последовательностях хранятся ссылки на ОБЪЕКТЫ ЛЮБОГО типа
		( → можно хранить элементы разных типов)
		
	► плоские (str, bytes, bytearray, memoryview, array.array)
		в плоских последовательностях хранятся сами значения
		в них можно хранить элементы только одного типа
	
	Плоские последовательности компактнее, но могут содержать только значения примитивных типов:
		- символы
		- байты
		- числа
	
	Последовательности:
		• изменяемые
		  (list, bytearray, array.array, collections.deque memoryview)
		• неизменяемые
		  (tuple, str, bytes)
