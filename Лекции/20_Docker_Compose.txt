Docker Compose. Оркестрация

EXPOSE - сообщает Docker engine о том, что в данном контейнере будет существовать процесс, прослушивающий заданный порт или несколько портов
При запуске контейнера нужно указать маппинг портов хоста и контейнера
docker run -p}{HOST_PORT}:{CONTAINER_PORT} image
Что за опция -P - динамический выбор порта?

Bind mounts - данные хранятся в произвольном месте файловой системы
Volumes - часть файловой системы хоста, управляемой Docker
tmpfs mounts - данные хранятся в RAM хоста
